# Top-Troops
Ever wanted to see the top 5 troops in Gorilla Tag? You can now! Unfortunately since this never made it into being an official game feature, it seems that they might not be very accurate to what the actual top troops are.

## How to install
Like any other mod, download the TopTroops.dll file from the Releases tab and put it into your plugins folder.
This mod is made for the **bepinex** mod loader.
