# Top-Troops
Ever wanted to see the top 5 troops in Gorilla Tag? You can now!

Unfortunately since this never made it into being an official game feature, it seems that they might not be very accurate to what the actual top troops are.

You can view the top 5 troops on the "TROOP" tab of the gorilla computer.

![Preview](preview.png)

## How to install
Like any other mod, download the TopTroops.dll file from the [Releases](https://github.com/lewis6777/Top-Troops/releases/latest) tab and put it into your plugins folder.
This mod is made for the **bepinex** mod loader.
